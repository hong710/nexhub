What are Categories?

Categories group servers by device type (e.g., Server, Desktop, Laptop, Embedded). Each server is automatically assigned to a category based on its hardware detection.

How to Create

  1. Click "Add category" button
  2. Enter the device type name (e.g., "Server", "Workstation")
  3. Click "Save"

Use Cases

  • Filter servers by type in the inventory
  • Apply different policies per category
  • Organize and monitor device fleet

Note: Categories are automatically created when the agent detects a new device type. You only need to manually add categories for custom classifications.

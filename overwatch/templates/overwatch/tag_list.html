What are Tags?

Tags are reusable labels you can attach to servers to organize, classify, and mark them. Unlike categories which are automatic, tags are manually applied for custom organization.

How to Create

  1. Click "Add tag" button
  2. Enter tag name (e.g., "Production", "Testing", "Backup")
  3. Add optional description
  4. Click "Save"

Tag Examples

  • Environment: Production, Staging, Development
  • Team: DevOps, Engineering, Support
  • Status: Critical, Monitored, Legacy

How to Use

Once created, tags can be assigned to servers in the server edit form. Use tags to quickly filter and identify related servers in your inventory.
